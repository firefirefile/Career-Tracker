# Career Tracker

Приложение для отслеживания карьерных откликов. Умеет: парсить вакансии с HH по ссылке, сохранять данные в таблицу, менять статусы отклика (жду, отказ, собес, архив). Это с натяжкой рабочая версия - планирую добавить возможность редактирования на странице вакансии и изменить дизайн.

## Структура проекта

- `todo-frontend/` - React + TypeScript фронтенд (Vite)
- `express-backend/` - Express.js бэкенд API
- `main.js` - Electron главный процесс

## Быстрый старт

```bash
# Установка зависимостей
npm run install-all

# Запуск в режиме разработки
npm run dev
```

## Сборка

```bash
npm run build
```

Приложение будет собрано в папке `dist/`.
