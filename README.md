# Career Tracker

Полноценное приложение для отслеживания карьерных откликов. 

## Структура проекта

- `todo-frontend/` - React + TypeScript фронтенд (Vite)
- `express-backend/` - Express.js бэкенд API
- `main.js` - Electron главный процесс

## Быстрый старт

```bash
# Установка зависимостей
npm run install-all

# Запуск в режиме разработки
npm run dev
```

## Сборка

```bash
npm run build
```

Приложение будет собрано в папке `dist/`.
