# Career Tracker - Frontend

React + TypeScript + Vite интерфейс для отслеживания вакансий.

## Технологии

- React 19
- TypeScript
- Vite
- React Router

## Скрипты

```bash
npm run dev      # Запуск dev-сервера
npm run build    # Сборка для продакшена
npm run preview  # Просмотр сборки
npm run lint     # Проверка кода
```

## API

Фронтенд взаимодействует с бэкендом по адресу `http://127.0.0.1:3000/api`.
